Welcome to Roberto's Recipe Book!

The app was developed with Angular connecting in a Firebase (Realtime Database - NoSQL).

Features:
    - Website developed in Angular 7;
    - Single Page Application (SPA);
    - Bootstrap;
    - Firebase Realtime Database (NoSQL) - Cloud solution;
    - Firebase email/password authentication;
    - Authorization per user;
    - Form using Reactive approach for dynamic ingredients controls;
    - Forms using Template-Driven approach for other components;
    - Application published into AWS S3;

Manual, a LOGGED user can:
    - See all the recipes from all users.
    - Add a new recipe.
    - Edit and delete your own recipes.
    - Create and see your own shopping-list.
